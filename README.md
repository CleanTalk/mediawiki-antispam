mediawiki-antispam
================
Version 2.3
================

**Requires:**

MediaWiki >= 1.29.0

**Installation:**

1.Download and place the file(s) in a directory called "Antispam" in your `extensions/` folder.
2. Add the following code at the bottom of your LocalSettings.php:

`wfLoadExtension( 'Antispam' );`
